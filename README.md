# Workout Boost | API
> REST API - Typescript, Express, Mongodb

## Quick Start
```
# Install dependencies
npm install

# Start Development
npm run dev

# Build API
npm run build

# Start API
npm run start
```
### Make a keys.ts in /config
```
module.exports = {
    database: "MONGO DB KEY FOR CONFIG",
    appSecret: "APP SECRET (anything you want)"
}
```
